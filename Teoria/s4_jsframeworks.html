1. Evolució de JavaScript al Client

Al principi, JS era caòtic i cada navegador tenia regles diferents per manipular el DOM. Les llibreries van néixer per unificar-ho tot fins arribar als potents frameworks actuals.

2. Els Pioners: Prototype.js i Script.aculo.us

Prototype.js

Una llibreria creada per dotar JavaScript de Programació Orientada a Objectes i facilitar crides AJAX.

  • Va inventar la funció màgica $() per substituir getElementById().
  • Extenia objectes natius (una mala pràctica avui dia, però revolucionari aleshores).

Script.aculo.us

Un complement construït sobre Prototype exclusivament per fer interfícies d'usuari (UI) dinàmiques.

  • Sistema Drag & Drop (arrossegar elements).
  • Efectes visuals complexos (Fade, Puff, Shake).
// Efecte Fade amb Script.aculo.us
new Effect.Fade('element_id', { duration: 2.0 });

3. jQuery: La Revolució Clàssica

Amb el lema "Write less, do more", jQuery va solucionar les incompatibilitats entre navegadors unificant-ho tot sota l'objecte jQuery o l'àlies $.

Els seus grans èxits:

  • Selecció amb selectors CSS (abans que existís querySelector).
  • Encadenament de mètodes (Method chaining).
  • L'event unificat $(document).ready().

Codi Clàssic

// Espera que el DOM estigui carregat
$(document).ready(function() {
  
  // Assigna event a un botó
  $("#boto_menu").click(function() {
    
    // Selecciona elements, canvia CSS i anima
    $(".panel")
        .css("background", "#333")
        .slideToggle("fast");
  });
  
});

4. És necessari jQuery a l'Actualitat?

VIGILA! Actualment, JS "Vanilla" natiu ha copiat gairebé totes les millores de jQuery. No s'hauria de carregar tota la llibreria per fer coses bàsiques.

On s'utilitza encara?

  • Projectes corporatius antics (Legacy).
  • L'ecosistema WordPress (themes/plugins).
  • Plantilles basades en Bootstrap 3 i 4.

jQuery vs Vanilla JS (Actual)

// JQUERY (Ocultar elements)
$(".caixa").hide();

// VANILLA JS MODERN
document.querySelectorAll('.caixa').forEach(el => {
  el.style.display = 'none';
});

// AJAX en JQUERY vs VANILLA
$.ajax({ url: 'api/dades' });
fetch('api/dades');

5. L'Era dels Frameworks Reactius (SPAs)

El canvi de paradigma

Abans, buscàvem elements a l'HTML i els modificàvem manualment (Manipulació Imperativa).

Ara, utilitzem l'enllaç de dades (Data Binding) orientat a components i reactivitat.

Single Page Applications (SPA)

Webs que carreguen l'HTML inicial un sol cop. El JS s'encarrega de redibuixar components sense recarregar la pàgina mai més.

Lògica Reactiva (Conceptes)

Només defineixes l'estat (les dades), i l'HTML s'actualitza automàticament on toqui.

<!-- Exemple conceptual Vue.js -->
<div>
  <h1>Hola {{ nomUsuari }}</h1>

  <!-- L'input modifica 'nomUsuari' en temps real -->
  <input v-model="nomUsuari" />
</div>

Ja no cal afegir EventListeners manuals ni fer innerHTML!
